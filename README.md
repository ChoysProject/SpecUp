# SpecUp
스터디 모임 지원 및 자격증 정보 제공 시스템 

Backend : JAVA, SpringBoot
Frontend : NEXT.js, React, Typescript
Server : AWS EC2, nginx
DB : AWS RDS, MongoDB
Build tool : Maven

------------------------------

명령어 정리
<frontend>
npm run dev
<backend>
mvn spring-boot:run
